# Мозговой штурм

Генерация и структуризация идей.

## Что это?

Креативная сессия для:
- Генерации новых идей
- Исследования альтернатив
- Решения сложных проблем

## Когда использовать?

- Не знаешь с чего начать
- Нужно придумать фичи
- Решаешь архитектурную проблему

## Как запустить?

```bash
/advanced:brainstorm "тема для обсуждения"
```

## Техники

1. **6 шляп мышления** — разные perspective
2. **SCAMPER** — модификация существующего
3. **Mind Map** — визуальные связи
4. **Worst Idea** — от плохих к хорошим

## Результат

Структурированный список идей с оценкой feasibility.
