# Исследование

Глубокое изучение темы или технологии.

## Что это?

Систематический сбор информации:
- Best practices
- Существующие решения
- Технические ограничения
- Примеры реализации

## Когда использовать?

- Выбираешь технологию
- Изучаешь новый домен
- Ищешь паттерны реализации

## Как запустить?

```bash
/advanced:research "тема исследования"
```

## Источники

- Официальная документация
- GitHub репозитории
- Stack Overflow
- Технические блоги

## Результат

Документ `docs/research-{topic}-{date}.md` с:
- Краткий обзор темы
- Альтернативы и сравнение
- Рекомендации
- Ссылки на источники
